# Ozon Orders Finder

Tampermonkey userscript для фильтрации заказов на странице архива Ozon по диапазону цены.

## Установка

1. Установите расширение [Tampermonkey](https://www.tampermonkey.net/) в браузер
2. Создайте новый скрипт и вставьте содержимое файла `ozon-orders-finder.user.js`
3. Откройте страницу архива заказов: `https://www.ozon.ru/my/orderlist?selectedTab=archive`

## Использование

- Введите минимальную и/или максимальную цену в рублях
- Нажмите **Найти** — подходящие заказы подсветятся зеленой рамкой
- При желании включите **Скрыть не подходящие** — остальные карточки скроются
- Нажмите **Сбросить**, чтобы убрать выделение и показать все заказы

## Особенности

- Автоматически применяет фильтр к динамически подгружаемым заказам при скролле
- Сохраняет настройки фильтра в `localStorage`
- При возврате на страницу фильтр применяется только если был активен до ухода
